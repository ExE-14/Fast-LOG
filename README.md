# Fast LOG (v1.2-MAX)

**Author:** BulkGecko6535 (notGecko)
*Lightweight Fabric mod for automated command execution on Minecraft 1.21.1.*

---

### Key Features

* **Auto-Command:** Automatically sends a specified command (e.g., `/login`) upon joining a server.
* **Smart Conditions:** Triggers only on specified IP addresses and optional status effects (like Blindness).
* **Chat Filtering:** Moves specified authentication plugin messages from the main chat to the Action Bar.
* **Visual Feedback:** Spawns a custom particle spiral and plays a sound effect upon successful execution.

### Configuration

1. Install **Cloth Config API** and **ModMenu**.
2. Access the settings via the ModMenu list or use the `/flog` command.
3. Set the execution delay (default is 40 ticks) and manage the Allowed IPs list.

### Installation

1. Place the `.jar` file into your `mods` folder.
2. Ensure you are running **Fabric Loader** for version 1.21.1.
